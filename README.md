Ansible Role: Kernel Settings
=========

Ansible role to configure kernel settings on Linux Servers.

Requirements
------------

The role does not require anything to run on RHEL and its derivatives.

Dependencies
------------

None.

Example Playbook
----------------

``` yaml
    - hosts: servers
      roles:
         - role: mikepruett3.kernel-settings
```
